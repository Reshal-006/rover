# ADR-005: Streamlit Dashboard for User Interface

## Status
Accepted

## Context
Rover needs an interactive visual dashboard for developers to paste repository URLs, launch scans, watch progress in real-time, view vulnerabilities, and click to fix issues. The UI must be fast to build, simple to maintain, and visually engaging.

## Decision
Use Streamlit to implement the visual UI. Apply customized Outfit typography and custom CSS/HTML glassmorphism styling to deliver a modern SaaS appearance.

## Consequences
Positive:
- Extremely rapid development of interactive components (tabs, cards, forms) using pure Python.
- Seamless updates and reactive data flows.
- Out-of-the-box styling overrides allow us to bypass generic browser default layouts.

Negative:
- Highly customized frontend behaviors require injecting raw HTML/CSS strings into markdown widgets.
- Streamlit's execution model re-runs the entire script on user interaction, requiring careful caching of history and scans.
