# Rover Deployment Guide

This document describes how to deploy the Rover system (both the FastAPI background listener and the Streamlit dashboard) to production.

---

## 1. FastAPI Backend Listener (Render)

The FastAPI webhook listener must run continuously to receive incoming events (e.g. issues, issue comments) from GitHub and run background scans/fixes.

### Deployment steps:
1. Sign up/log in to [Render](https://render.com/).
2. Create a new **Web Service** and connect your public GitHub fork of Rover.
3. Configure the service settings:
   - **Runtime**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn api.main:app --host 0.0.0.0 --port $PORT`
4. Add the following **Environment Variables** in the Render settings pane:
   - `GEMINI_API_KEY`: Your Gemini API credential key.
   - `USE_GITHUB_APP`: `true`
   - `GITHUB_APP_ID`: Your GitHub App ID.
   - `GITHUB_PRIVATE_KEY`: The raw private key PEM string (copy-paste directly, including headers).
   - `GITHUB_WEBHOOK_SECRET`: Your webhook secret.
   - `GITHUB_CLIENT_ID`: Your GitHub App Client ID.
   - `GITHUB_CLIENT_SECRET`: Your GitHub App Client Secret.
5. Deploy the service. Take note of the public HTTPS URL generated by Render.

---

## 2. GitHub App Webhook Integration

Once you have deployed the FastAPI backend on Render, you must configure the GitHub App to route events to it:
1. Navigate to your GitHub App developer settings.
2. Update the **Webhook URL** field to point to your Render service webhook path:
   `https://your-render-service.onrender.com/webhook`
3. Save changes. GitHub will now send webhooks directly to Render!

---

## 3. Streamlit Dashboard (Streamlit Cloud or Render)

You can deploy the interactive user interface to Streamlit Community Cloud for free.

### Deployment steps:
1. Sign in to [Streamlit Share](https://share.streamlit.io/).
2. Click **New App** and select your repository, branch, and entrypoint path:
   - **Repository**: `your-username/rover`
   - **Branch**: `main`
   - **Main file path**: `dashboard/app.py`
3. Click **Advanced settings** and paste your environment configurations:
   ```toml
   USE_GITHUB_APP = "true"
   GITHUB_APP_ID = "123456"
   GITHUB_PRIVATE_KEY = """-----BEGIN RSA PRIVATE KEY-----
   ...
   -----END RSA PRIVATE KEY-----"""
   GITHUB_WEBHOOK_SECRET = "your_secret"
   GITHUB_CLIENT_ID = "your_client_id"
   GITHUB_CLIENT_SECRET = "your_client_secret"
   GEMINI_API_KEY = "your_gemini_key"
   ```
4. Click **Deploy**. Streamlit Cloud will build and serve the application!
